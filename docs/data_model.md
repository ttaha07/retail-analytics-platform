# Retail Analytics Data Model

## Star Schema

### Fact Table

FACT_SALES

Measures:

- PRICE
- FREIGHT_VALUE
- TOTAL_SALE_AMOUNT

### Dimensions

DIM_CUSTOMER

Attributes:

- CUSTOMER_ID
- CUSTOMER_UNIQUE_ID
- CUSTOMER_CITY
- CUSTOMER_STATE

DIM_PRODUCT

Attributes:

- PRODUCT_ID
- PRODUCT_CATEGORY_NAME
- PRODUCT_WEIGHT_G

DIM_DATE

Attributes:

- ORDER_DATE

## Relationships

FACT_SALES.CUSTOMER_ID → DIM_CUSTOMER.CUSTOMER_ID

FACT_SALES.PRODUCT_ID → DIM_PRODUCT.PRODUCT_ID

FACT_SALES.ORDER_DATE → DIM_DATE.ORDER_DATE
